## Installation
1.)PyQt5
```
PyQt5 >= 5.15.9
```
2.)pytorch
```
Please install throught official website of pytorch
torch               1.13.1+cu116
torchaudio          0.13.1+cu116
torchvision         0.14.1+cu116
```
3.)dilib
```
dlib == 19.22.99
```
4.)Opencv
```
pip install opencv-python == 4.8.0.74
```
